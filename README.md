## To Run

1. clone repo ```git@github.com:Asante-Adarkwa-Usman/MAD-Js-Basics.git```
2. run ```cd MAD-Todo-Backend``` 
3. run ```npm install```
4. run ```nodemon index.js```(for mac/linux) and ```npx nodemon start```(for windows)
